# SCT_DA_2 - Data Cleaning and Preparation

## 📌 Task Description
Load a dataset into a Python environment using the Pandas 
library, identify and handle missing values, drop duplicate 
rows, and convert data types. Export the cleaned data 
to a new CSV file.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Google Colab

## 📂 Dataset
Superstore Sales Dataset (train.csv)
- 9,800 rows
- 18 columns

## ✅ Steps Performed
- Loaded dataset using Pandas
- Explored data using df.info() and df.describe()
- Handled missing values in Postal Code column
- Removed duplicate rows
- Converted Order Date & Ship Date to datetime format
- Created new Shipping Days column
- Exported cleaned data as cleaned_superstore.csv

## 💡 Key Insights
- Only 11 missing values found in Postal Code column
- Average shipping time is approximately 4 days
- Fastest shipment is same day (0 days)
- Maximum shipping time is 7 days

- cleaned_superstore.csv - Cleaned dataset
